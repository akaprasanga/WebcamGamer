    The Process

    Our product uses an object detection model called YOLO. YOLO utilizes a system of Convolutional Neural Networks (CNN), which is a deep learning algorithm that is used in image recognition and processing and is specifically designed to process pixel data. With YOLO and tedious training, we were able to detect various hand gestures (Right, Left, Up, Stop, and Attack) into classes. With YOLO's live image detection, we can connect our model to a live-footage source, such as a webcam, and use the classes

    Motives

    What motivated us to work on this project

    We wanted to see if we could find a way to combine both game development and the lessons of AI Camp.

    We wanted to do something with hand-tracking.

    We figured that this project would be really beneficial for us as it was a challenging one, meaning that we would learn a lot of stuff as we worked on it.

    We knew we would face a bunch of roadblocks such as the time constraint and finding a way to connect the hand gesture controls to a game.
